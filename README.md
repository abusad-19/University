# University Management System

## Overview
A full-stack web application for managing university administrative requests and certificate issuance. Built with a clean 3-layer architecture (BLL, DAL, MVC) to ensure scalability and maintainability.

## Key Features
- **Request Management**: Submit, cancel, approve, or reject requests.
- **Certificate Lifecycle**: Automatically generate certificates upon approval.
- **Role-Based Authorization**: Secure access for applicants and administrators.
- **Layered Architecture**: Clear separation of Business Logic, Data Access, and Presentation.

## Technologies Used
- **Backend**: C#, ASP.NET MVC, Entity Framework
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQL Server
- **Architecture**: 3-Layer (BLL, DAL, MVC)

## Project Structure
- **`University.BLL`**: Business Logic Layer (core application rules).
- **`University.DAL`**: Data Access Layer (database interactions using Entity Framework).
- **`University.MVC`**: Presentation Layer (UI and user interaction).

## Setup & Installation
1.  Clone the repository.
2.  Open the solution in Visual Studio.
3.  Update the database connection string in `appsettings.json`.
4.  Run the database migrations (if applicable).
5.  Build and run the application.
